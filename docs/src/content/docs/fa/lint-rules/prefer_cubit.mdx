استفاده از نمونه‌های `Cubit` را به‌جای نمونه‌های `Bloc` ترجیح دهید.

## دلیل

این قانون صرفاً یک قاعدهٔ سبک است. در برخی موارد، تیم‌ها ممکن است ترجیح دهند
برای حفظ یکپارچگی و سازگاری، در سراسر برنامه فقط از نمونه‌های `Cubit` استفاده
کنند.

:::tip

برای آشنایی بیشتر با مزایای `Cubit` به
[Core Concepts](/fa/bloc-concepts/#cubit-مزایای) مراجعه کنید.

:::
