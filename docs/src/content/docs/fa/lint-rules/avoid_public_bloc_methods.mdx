از در معرض قرار دادن متدهای عمومی در نمونه‌های `Bloc` خودداری کنید.

## دلیل

Blocها به رویدادهای ورودی واکنش نشان می‌دهند و وضعیت‌های خروجی را منتشر می‌کنند. بنابراین
روش پیشنهادی برای ارتباط با یک نمونهٔ bloc استفاده از متد `add` است. در اکثر موارد نیازی
به ایجاد انتزاع‌های اضافی روی API `add` نیست.
