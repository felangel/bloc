استفاده از نمونه‌های `Bloc` را به‌جای نمونه‌های `Cubit` ترجیح دهید.

## دلیل

این قانون صرفاً یک قاعدهٔ سبک است. در برخی موارد تیم‌ها ممکن است ترجیح دهند تا
برای حفظ یکپارچگی و سازگاری، در سراسر برنامه فقط از نمونه‌های `Bloc` استفاده
کنند.

:::tip

برای آشنایی بیشتر با مزایای `Bloc` به
[Core Concepts](/fa/bloc-concepts/#مزایای-bloc) مراجعه کنید.

:::
