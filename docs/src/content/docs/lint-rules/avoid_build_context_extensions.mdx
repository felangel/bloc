Avoid using `BuildContext` extensions to access `Bloc` or `Cubit` instances.

## Rationale

Prefer using `BlocProvider.of<T>(context)`, `BlocBuilder`, and `BlocSelector`
over `BuildContext` extensions like `context.read<T>()`, `context.watch<T>()`,
and `context.select<T, R>(...)` for more explicit code and to enforce
consistency.
