`Cubit` örnekleri yerine `Bloc` örneklerini tercih edin.

## Gerekçe

Bu kural tamamen stilistik bir kuraldır. Bazı durumlarda, ekipler tutarlılık
adına uygulamalarının tamamında yalnızca `Bloc` örneklerini kullanma
standardını benimsemeyi tercih edebilir.

:::tip

`Bloc`'un avantajları hakkında daha fazla bilgi edinmek için
[Temel Kavramlar](/bloc-concepts/#bloc-advantages) bölümüne bakın.

:::
