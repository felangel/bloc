`Cubit` örnekleri yerine `Bloc` örneklerini kullanmayı tercih edin.

## Neden?

Bu kural tamamen stilistik bir kuraldır. Bazı durumlarda ekipler, tutarlılık
adına uygulamalarının tamamında yalnızca `Bloc` örneklerini kullanmayı
standartlaştırmak isteyebilir.

:::tip

`Bloc`'un avantajları hakkında daha fazla bilgi için
[Temel Kavramlar](/tr/bloc-concepts/#blocun-avantajları) bölümüne göz atın.

:::
