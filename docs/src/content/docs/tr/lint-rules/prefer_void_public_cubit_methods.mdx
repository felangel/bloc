`Cubit` örnekleri üzerinde void genel metotları tercih edin.

:::note

Bu lint kuralı [`package:bloc_lint`](https://pub.dev/packages/bloc_lint)
paketinin `0.2.0-dev.2` sürümünde tanıtılmıştır.

:::

## Neden?

`Cubit` örnekleri üzerindeki genel metotlar; `Cubit`'i bilgilendirmek ve `emit`
metodu aracılığıyla durum değişikliklerini başlatmak için kullanılmalıdır.
Çağıran tarafın herhangi bir durum bilgisine erişmesi gerekiyorsa bunu doğrudan
`state` üzerinden yapmalıdır.

:::note

Aşağıdaki kurallar ilgilidir ve genellikle `prefer_void_public_cubit_methods`
ile birlikte etkinleştirilir.

- [`avoid_public_bloc_methods`](/tr/lint-rules/avoid_public_bloc_methods)
- [`avoid_public_fields`](/tr/lint-rules/avoid_public_fields)

:::
