Dosya adlandırma kurallarına uymayı tercih edin.

:::note

Bu lint kuralı [`package:bloc_lint`](https://pub.dev/packages/bloc_lint)
paketinin `0.3.0` sürümünde tanıtılmıştır.

:::

## Neden?

Tutarlılık, bakım kolaylığı ve sorumlulukların ayrılması için bloc ve cubit
örneklerini inline tanımlamak yerine kendi Dart dosyalarında tanımlamayı tercih
edin.

:::tip

Yeni bloc/cubit örneklerini hızlı ve tutarlı şekilde oluşturmak için
[package:bloc_tools](https://pub.dev/packages/bloc_tools) paketinin
`bloc new <component>` komutunu kullanmayı düşünün.

:::
