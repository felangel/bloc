`Bloc` örnekleri yerine `Cubit` örneklerini kullanmayı tercih edin.

## Neden?

Bu kural tamamen stilistik bir kuraldır. Bazı durumlarda ekipler, tutarlılık
adına uygulamalarının tamamında yalnızca `Cubit` örneklerini kullanmayı
standartlaştırmak isteyebilir.

:::tip

`Cubit`'in avantajları hakkında daha fazla bilgi için
[Temel Kavramlar](/tr/bloc-concepts/#cubitin-avantajları) bölümüne göz atın.

:::
