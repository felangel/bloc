Bir `Bloc` veya `Repository` örneğine erişmek için `BuildContext` uzantılarını
kullanmayı tercih edin.

:::note

Bu lint kuralı [`package:bloc_lint`](https://pub.dev/packages/bloc_lint)
paketinin `0.3.2` sürümünde tanıtılmıştır.

:::

## Neden?

Tutarlılık adına, `BlocProvider.of`, `RepositoryProvider.of`, `BlocBuilder` veya
`BlocSelector` yerine `context.read`, `context.watch` ve `context.select` gibi
`BuildContext` uzantılarını kullanmayı tercih edin.

| açık metot                                                            | uzantı                |
| --------------------------------------------------------------------- | --------------------- |
| `BlocProvider.of<Bloc>(context, listen: false)`                       | `context.read<Bloc>`  |
| `BlocBuilder<Bloc, State>(...)` veya `BlocProvider.of<Bloc>(context)` | `context.watch<Bloc>` |
| `BlocSelector<Bloc, State>(...)`                                      | `context.select`      |
