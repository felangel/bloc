`Bloc` veya `Cubit` örneklerine erişmek için `BuildContext` uzantılarını
kullanmaktan kaçının.

:::note

Bu lint kuralı [`package:bloc_lint`](https://pub.dev/packages/bloc_lint)
paketinin `0.3.0` sürümünde tanıtılmıştır.

:::

## Neden?

Tutarlılık ve açıklık adına, `BuildContext` uzantıları yerine doğrudan ilgili
metotları kullanmayı tercih edin. Bu yaklaşım test edilebilirlik açısından da
faydalıdır çünkü uzantı (extension) metotlarını mocklamak mümkün değildir.

| uzantı           | açık metot                                                            |
| ---------------- | --------------------------------------------------------------------- |
| `context.read`   | `BlocProvider.of<Bloc>(context, listen: false)`                       |
| `context.watch`  | `BlocBuilder<Bloc, State>(...)` veya `BlocProvider.of<Bloc>(context)` |
| `context.select` | `BlocSelector<Bloc, State>(...)`                                      |
