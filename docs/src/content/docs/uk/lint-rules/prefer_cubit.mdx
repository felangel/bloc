Надавайте перевагу використанню екземплярів `Cubit` замість екземплярів `Bloc`.

## Обґрунтування

Це правило є суто стилістичним. У деяких випадках команди можуть надавати
перевагу стандартизації використання лише екземплярів `Cubit` у всьому
застосунку заради узгодженості.

:::tip

Дізнайтеся більше про переваги `Cubit` у розділі
[Основні концепції](/uk/bloc-concepts/#переваги-cubit).

:::
