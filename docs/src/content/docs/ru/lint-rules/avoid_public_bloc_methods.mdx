Избегайте предоставления публичных методов в экземплярах `Bloc`.

## Обоснование

Блоки реагируют на входящие события и выдают исходящие состояния. В результате
рекомендуемый способ взаимодействия с экземпляром bloc — через метод `add`.
В большинстве случаев нет необходимости создавать дополнительные абстракции поверх
API `add`.
