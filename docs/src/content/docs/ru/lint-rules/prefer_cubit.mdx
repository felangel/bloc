Предпочитайте использование экземпляров `Cubit` вместо экземпляров `Bloc`.

## Обоснование

Это правило является чисто стилистическим. В некоторых случаях команды могут
предпочесть стандартизировать использование только экземпляров `Cubit` во всем
приложении для согласованности.

:::tip

Узнайте больше о преимуществах `Cubit` в разделе
[Основные концепции](/bloc-concepts/#cubit-advantages).

:::
