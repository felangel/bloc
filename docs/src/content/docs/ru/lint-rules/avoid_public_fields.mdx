Избегайте предоставления публичных полей в экземплярах `Bloc` и `Cubit`.

## Обоснование

Компоненты бизнес-логики поддерживают свое собственное `state` и выдают изменения состояния через
API `emit`. В результате все публично доступное состояние должно быть предоставлено через
объект `state`.

**НЕ ПРЕДОСТАВЛЯЙТЕ** публичные поля в экземплярах bloc и cubit.
