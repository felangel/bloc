Предпочитайте использование экземпляров `Bloc` вместо экземпляров `Cubit`.

## Обоснование

Это правило является чисто стилистическим. В некоторых случаях команды могут
предпочесть стандартизировать использование только экземпляров `Bloc` во всем
приложении для согласованности.

:::tip

Узнайте больше о преимуществах `Bloc` в разделе
[Основные концепции](/bloc-concepts/#bloc-advantages).

:::
